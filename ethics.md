# Ethics, Privacy, and Data Governance

This project is intended for research and educational purposes only and is **not** a clinical tool.

## Data Sources
- This repository does **not** store or distribute any private patient data.
- If restricted clinical datasets (e.g., MIMIC-IV) are used locally, they must be accessed through approved channels and remain on the userâ€™s machine.
- To ensure reproducibility and public accessibility, the repository supports a synthetic dataset option.

## Privacy & Safety
- No personally identifiable information (PII) is included in this repository.
- Outputs are designed for decision support research and do not make diagnostic claims.
- Model outputs should be interpreted with uncertainty and human oversight.

## Responsible Use
- This project is not validated for clinical deployment.
- Any clinical use would require institutional review, validation, and governance approvals.

## License & Compliance
Users must comply with the dataset license(s) and access agreements of any external datasets they use (e.g., PhysioNet credentialing requirements).
